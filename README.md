# GTC(Growtopaia Community) Project
그로우토피아 게임의 웹 커뮤니티 프로젝트입니다.
게임에 거래가 주목적인 웹 페이지이고 react-native 로 앱도 구성예정입니다.

## 개발환경

- node v10.16.0
- MariaDB
- CRAv2 (create-react-app v2)
- IDE ( WebStorm )
